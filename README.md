# RPGgame_java
## 介紹
這個遊戲是一個結合了戰鬥以及pokemon劇情的RPG遊戲，有多個戰鬥地圖，玩家可以選擇自己喜歡的職業，用金錢購買更強的武器打怪升級，擊敗最後的大魔王之後破關
